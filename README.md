# Scientific Calculator (Python Tkinter)

A scientific calculator built using Python and Tkinter with a simple graphical user interface (GUI). It supports basic arithmetic operations as well as common scientific functions.

## Features

- Basic arithmetic operations (+, -, ×, ÷)
- Decimal number support
- Clear (C) button
- Scientific functions:
  - sin()
  - cos()
  - tan()
  - square root (√)
  - log (base 10)
  - natural logarithm (ln)
  - π (Pi)
  - e (Euler's Number)
  - x² (Square)
  - x³ (Cube)
- Degree (DEG) and Radian (RAD) mode switching
- Simple and user-friendly GUI using Tkinter

## Technologies Used

- Python
- Tkinter
- Math Module

## How to Run

1. Make sure Python is installed.
2. Save the file as `scientific_calculator.py`.
3. Run the following command:

```bash
python scientific_calculator.py
```
